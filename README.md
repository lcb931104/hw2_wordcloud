# hw2_wordcloud 建立文字雲GUI
## 功能
### 1. 英文文字雲

- 輸入英文或讀入純英文的文字檔(.text)
- 選擇輸入的內容裡面要有多少單詞放入文字雲
- 選擇文字雲的主題色
- 生成文字雲

### 2. 中文文字雲

- 輸入中文或讀入文字檔(.text)，可以有英文但不能全是英文
- 選擇輸入的內容裡面要有多少單詞放入文字雲
- 選擇文字雲的主題色
- 生成文字雲
